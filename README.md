# UwUrg

a port of [turg]() from C to C++ with OOP for `OOP Class` Project.

## Build
__Build:__

```sh
mkdir build
cd build
cmake -DCMAKE_BUILD_TYPE=Release .. #or `Debug` for debug build.
make j`nproc`
```
