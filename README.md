# UwUrg

a port of [turg](https://github.com/commrade-goad/turg) from C to C++ with for `OOP Class` Project.

## Library used
- [raylib](https://github.com/raysan5/raylib)

## Build

```sh
mkdir build
cd build
cmake -DCMAKE_BUILD_TYPE=Release .. #or `Debug` for debug build.
make -j`nproc`
```
